## 2.模块拆分
模块大致可分为4个模块:产品列表、商品信息、购物车、订单<br>

1.产品列表主要是展示各种各样的商品供用户选择，也可通过查询功能查找商品<br>
2.商品信息是介绍某一商品的各种信息，比如价格、所属店铺等等，用户可以加入购物车或者购买<br>
3.购物车即是用户加入购物车且未下单的所有商品，可以选择下单，也可以删除<br>
4.订单包括用户已经收货的历史订单和未收货的订单以及未付款的订单<br>
